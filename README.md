# YouBike 站點搜尋
**一個簡單的YouBike站點搜尋爬蟲**

---

**簡介**

此專案為一個使用Python編寫的YouBike搜尋器，提供使用者可以搜尋全國的 YouBike 站點資訊，並查看各站點的車輛狀況。

使用者可以透過輸入站點名稱或定位，快速查詢 YouBike 2.0 站點的資訊，包括：
* 站點名稱
* 距離
* 地區
* 地址
* YouBike 2.0 數量
* YouBike 2.0E 數量
* 可停空位數

## 使用方式

1. 將上方檔案下載至裝置運行或是前往[線上演示網頁](https://potatosserver.github.io/YouBike_Python/)執行程式碼即可使用本服務。
